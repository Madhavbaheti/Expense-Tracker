# Expense Tracker - Personal Finance Management

Expense Tracker is a web application that empowers users to manage their expenses seamlessly. Developed using React.js for the frontend, Express for the backend, and MongoDB for the database, it offers features such as real-time expense tracking, PDF generation, and a user-friendly interface.

## Features

- **User Registration and Authentication**: Create an account to personalize your expense tracking experience.
- **Card Management**: Add, edit, and delete cards for organizing expenses.
- **Expense Logging**: Record expenses with remarks and real-time date tracking.
- **Total Expenditure**: View the total expenditure for each card and the overall total.
- **Update and Delete Expenses**: Edit or remove already added expenses for better control over your records.
- **PDF Export**: Generate PDF reports of your expenses for easy record-keeping.

## Technologies Used

- **Frontend**: React.js
- **Backend**: Express
- **Database**: MongoDB
